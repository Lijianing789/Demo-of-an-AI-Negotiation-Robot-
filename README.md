# Demo-of-an-AI-Negotiation-Robot-
调用智谱API智能体搭建ai竞价客服。采用streamlit实现简单demo。理想产品应在购物app中能够调用多种商品信息，根据商家最低底价与C端用户进行竞价。用户能不断与ai客服竞价成交价，增加趣味性。
不断优化prompt，使得成交价不低于商家底价，但能在清仓价格上给予用户一定优惠，用户在聊天对话中砍价，购买意图上升。
